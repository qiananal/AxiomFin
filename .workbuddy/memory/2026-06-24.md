# 2026-06-24 AxiomFin 项目升级

## 升级概述
将 AxiomFin 从基础投研中台升级为完整的 **量价精选+买卖信号+持仓监控** 量化中台 v2.0。

## 新增模块

### 1. services/screener.py - 量价筛选引擎
- 从 55 只沪深热门股池中基于量价关系多因子打分
- 评分维度：当日量价关系(25分)、近期趋势(20分)、量能结构(20分)、突破形态(20分)、活跃度(15分)
- 支持批量实时行情 + K线深度分析
- 筛选结果持久化到 data/daily_picks.json

### 2. services/signal_analyzer.py - 买卖信号分析器
- 综合 MA5/10/20/60、RSI(14)、MACD、支撑阻力位
- 买入信号：放量突破均线、金叉、RSI超卖、缩量回踩支撑、阳包阴
- 卖出信号：放量跌破均线、死叉、RSI超买、高位放量滞涨、长上影
- 输出建议买入价和卖出价 + 持仓诊断
- 信号等级：STRONG_BUY/BUY/WEAK_BUY/HOLD/WEAK_SELL/SELL/STRONG_SELL

### 3. services/portfolio_monitor.py - 持仓监控模块
- 逐股调用 signal_analyzer 进行深度分析
- 自动判断补仓/卖出/风险提示
- 告警分级：1=提示 2=重要 3=紧急
- 重要告警自动发邮件
- 数据持久化到 data/portfolio_alerts.json

### 4. 前端升级 app_frontend.py v2.0
- Tab式布局：每日精选Top5 | 买卖信号分析 | 持仓诊断 | 智能投研
- 侧边栏：资产看板 + 监控任务 + 实时告警
- 后台线程1：多因子巡检(60s)
- 后台线程2：持仓监控(5min) + 每日精选刷新(30min)

### 5. Agent引擎扩展
- TOOLS_MAPPING 从6把扩展到11把武器
- Schema枚举同步更新
- 新增工具：screen_daily_top_stocks, analyze_stock_signals, get_portfolio_diagnosis, get_daily_picks_cached, get_portfolio_alerts

## 技术栈
- Python 3.13 (managed venv: .venv_managed)
- pytdx 连接通达信行情
- streamlit 1.58 前端
- DeepSeek API 智能体推理

## 启动方式
```
.venv_managed/Scripts/streamlit run app_frontend.py
```
